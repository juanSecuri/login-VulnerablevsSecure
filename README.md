# login-VulnerablevsSecure
Monté un laboratorio comparando un login vulnerable a SQL Injection y su versión segura. Automatizé pruebas E2E con Serenity BDD, generé reportes automáticos y orquesté la ejecución desde un agente self-hosted en Azure DevOps. Aprendí a parametrizar pipelines, ajustar WebDriver y validar mitigaciones.
